# Grupo Maxpesa — Site Institucional

React + Vite + CSS Modules. Tema claro, vermelho `#D91F2A`, tipografia Outfit + Inter.

## Estrutura

```
src/
  assets/          logo.png
  components/      Topbar, Header, Footer, WppButton, Eyebrow, Button,
                   SectionHeader, PageHero, Breadcrumb, ImgFrame, FeatList,
                   Timeline, EsgItem, BranchCard, ContactForm, Toast, Layout
  hooks/           useReveal.js  useCounter.js
  pages/           Home, Empresa, Servicos, Esg, Certificacoes, Contato
  styles/          global.css
  App.jsx          Router
  main.jsx         Entry point
```

## Setup

```bash
npm install
npm run dev
```
